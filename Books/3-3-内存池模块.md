## 3-3.内存池模块 - MemoryPool
使用案例
``` csharp
/// <summary>
/// 资源组数据。
/// <remarks>DisposeGroup。</remarks>
/// </summary>
public class AssetGroup : IMemory
{
    public void Clear(){}
}

//内存池中获取/生成内存对象。
AssetGroup assetGroup = MemoryPool.Acquire<AssetGroup>();

//释放内存对象还给内存池。
MemoryPool.Release(assetGroup);
```
